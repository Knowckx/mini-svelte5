# 快速初始化新项目
```bash
pnpm dlx degit Knowckx/mini-svelte5 yourAppName
cd yourAppName
pnpm install


# 初始化新的 git 仓库
git init
git add .
git commit -m "Initial commit from template"
git remote add origin your-repo-url
git push -u origin main
```

```bash
# 启动
pnpm dev

# 测试pwa模式
pnpm run build && pnpm preview --host
```
